## 📌 Project Overview

SpaceXplore is a modern and responsive Space Exploration Landing Page designed to inspire users to learn more about the universe, planets, galaxies, and historic space missions.

The website features an attractive space-themed design with glassmorphism effects, animated planets, floating astronaut visuals, mission timelines, and interactive sections that provide information about our solar system and space exploration.

---

## ✨ Features

* Responsive Navigation Bar
* Hero Section with Space Theme
* Floating Astronaut Animation
* Rotating Planet Animations
* About Space Exploration Section
* Solar System Planets Section
* Milky Way Galaxy Section
* Historic Space Missions Sectio
* Amazing Space Facts Section
* Contact Form
* Modern Glassmorphism UI Design
* Fully Responsive Layout
